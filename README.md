## To run in local on your device

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```
2. Add gatsby to the project

   ```sh
   yarn add gatsby
   ```

2. Once installed

   ```sh
   gatsby develop
   ```
   
## The project

This project is a simple portfolio realized in reactjs whith gatsby generator.
